## Overview
Project ADT (Adaptive Defense T-cell) represents a revolutionary architectural redesign of the human immune system. By treating biological defense as a distributed network security problem, we have developed a Two-Step Authentication Protocol for T-cells designed to eliminate zero-day viral pathogens while preventing cytokine storms.

## Key Technical Features
*   Decentralized Authentication (Edge Computing): Local token issuance via Dendritic Exosomes to bypass lymph node latency.
*   Whisper Mode Liquidation: Precision cellular clearance with 90% less inflammation than wild-type immune responses.
*   Metabolic Optimization: Engineered to minimize ATP consumption during chronic viral pressure.
*   Evolutionary Persistence: Adaptive recognition logic that keeps pace with rapid viral mutations (Influenza/Cold analogs).

## Scientific Validation Suite
The project includes a suite of high-fidelity simulations built with the Mesa Agent-Based Modeling framework:
*   apocalypse_simulation.py: Stress-testing under maximum viral aggression.

*   clinical_simulation.py: Modeling a Clean Handover protocol for severe SARS-CoV-2 infections.
*   high_res_analysis.py: Sub-macro-step latency tracking at the nanosecond equivalent scale.
